# agent-catalog

[![agent-catalog:ard-conformance](https://github.com/muredata/agent-catalog/actions/workflows/conformance.yml/badge.svg)](https://github.com/muredata/agent-catalog/actions/workflows/conformance.yml)

A catalog for Agentic Data Resources, implements [Agentic Resource Discovery (ARD) specification](https://agenticresourcediscovery.org/).


- Catalog: https://muredata.com/agent-catalog
- Registry: https://ard.muredata.com/v1/search

### Usage

Install Skill on Claude Code, Codex or GitHub Copilot:
```
curl -fsSL https://raw.githubusercontent.com/muredata/agent-catalog/main/install.sh | bash
```
